# AEO Recommend

Generate actionable recommendations for improving AI search visibility using [Goose AEO](https://github.com/athina-ai/goose-aeo). Analyzes your latest run data to identify gaps and opportunities.

## What It Does

1. Analyzes your latest AEO run results
2. Identifies visibility gaps (topics where your brand is missing)
3. Finds source opportunities (domains frequently cited by AI engines)
4. Provides competitor insights (what competitors are doing differently)
5. Offers to draft content, comparison pages, or outreach pitches

## Prerequisites

- Goose AEO must be initialized (use `/aeo-setup`)
- At least one completed run (use `/aeo-run`)

## Usage

```
/aeo-recommend
```
