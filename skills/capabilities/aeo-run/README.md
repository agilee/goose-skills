# AEO Run

Run an [AEO](https://github.com/athina-ai/goose-aeo) visibility analysis. Queries AI search engines with your saved queries, analyzes responses for brand mentions, and presents a conversational report.

## What It Does

1. Estimates cost before running (requires confirmation)
2. Queries AI engines (ChatGPT, Perplexity, Gemini, Grok, Claude, DeepSeek)
3. Analyzes responses for brand mentions, prominence, and sentiment
4. Generates a visibility report with insights and recommendations

## Prerequisites

- Goose AEO must be initialized first (use `/aeo-setup`)
- AI provider API keys configured in `.env`

## Usage

```
/aeo-run
```
